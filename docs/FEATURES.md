# RedMagicBox 功能文档

## 产品概述

RedMagicBox 是一款军事级加密安全隔离箱，采用 AES-256-CBC + PBKDF2-SHA256 + HMAC-SHA256 三重加密体系，所有运算均在本地完成，用户数据永远不会离开设备。

- **版本**: 3.9.14
- **平台**: Android
- **许可证**: RedMagicBox Personal Use License (RMPUL)
- **开源**: 半开源（GitHub: AHPNFT/RedMagicBox）

---

## 核心功能

### 1. 用户系统

| 功能 | 说明 |
|------|------|
| 注册 | 设置 RedID（用户名）+ 登录密码 |
| 登录 | RedID + 密码验证，支持生物识别 |
| 密码安全 | PBKDF2-SHA256 10万次迭代密钥推导 |
| 防截图 | FLAG_SECURE 标志，防止截屏/录屏 |
| Root 检测 | 检测设备 Root 状态，警告安全风险 |

### 2. 文件加密

- **加密算法**: AES-256-CBC
- **密钥推导**: PBKDF2-SHA256，100,000 次迭代
- **完整性校验**: HMAC-SHA256
- **加密引擎**: C++ Native SO 库（代码混淆）
- **支持模式**:
  - 文件加密：选择任意文件加密为 `.red` 格式
  - 文本加密：输入文本内容加密为 `.red` 格式
- **自定义文件名**: 加密时用户自定义文件名，输出为 `自定义名.red`
- **原文件处理**: 加密后可选删除原文件
- **加密次数限制**: 免费版 5 次，激活后无限次

### 3. 文件解密

- **解密验证**: 用户 RedID + 密码 + 激活码三重验证
- **跨设备解密**: 同一 RedID + 密码 + 激活码可在不同设备解密
- **完整性检查**: HMAC-SHA256 校验，防止文件被篡改
- **全盘扫描**: 扫描设备存储中所有 `.red` 文件
  - 递归扫描外部存储，最大深度 8 层
  - 实时显示扫描进度
  - Android 11+ 需「所有文件访问」权限
- **文件来源识别**: 自动识别 `.red` 文件的归属 RedID

### 4. 文件列表管理

- 查看所有已加密文件
- 支持解密、分享、删除操作
- 全盘扫描外部 `.red` 文件
- 显示文件大小、类型等元信息

### 5. 安全分享

- 选择加密文件通过系统分享功能发送
- 支持全盘扫描选择外部 `.red` 文件
- 分享时复制文件到缓存目录，不暴露原始路径
- 支持所有 Android 分享渠道（微信、QQ、邮件等）

### 6. 激活码系统

#### 激活码格式
```
XXXX-XXXX-XXXX-XXXX
```
16 位大写字母+数字，4 位一组，最后 4 位为 HMAC-SHA256 校验码。

#### 激活码作用
- 解锁无限次加密
- 参与加密密钥推导，是解密的必要条件
- 支持跨设备解密

#### 安全提示
- 激活码丢失将无法解密文件
- 泄露激活码有泄露加密文件的风险
- 请妥善保管激活码

## 技术架构

### 加密流程

```
原始文件 → Base64编码 → PBKDF2(RedID+密码+激活码, salt, 100000次) → AES-256-CBC加密 → HMAC-SHA256签名 → .red文件
```

### 解密流程

```
.red文件 → HMAC校验 → PBKDF2(RedID+密码+激活码, salt, 100000次) → AES-256-CBC解密 → Base64解码 → 原始文件
```

### Native 模块

| 模块 | 功能 |
|------|------|
| CryptoModule | C++ 加密/解密核心引擎 |
| StoragePermissionModule | Android 11+ 所有文件访问权限管理 |
| AppChooserModule | 原生应用选择器（钱包/浏览器） |
| FileOpenerModule | 文件打开功能 |

### 权限说明

| 权限 | 用途 |
|------|------|
| MANAGE_EXTERNAL_STORAGE | 全盘扫描 .red 文件（Android 11+） |
| READ_EXTERNAL_STORAGE | 读取外部存储文件 |
| WRITE_EXTERNAL_STORAGE | 保存加密文件 |
| INTERNET | 网络状态检测（加密本身不需要网络） |
| VIBRATE | 触觉反馈 |
